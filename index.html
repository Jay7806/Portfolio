NC-News

My NC-News project is a full-stack news app I built as my solo project at Northcoders. This was originally two seperate projects, a back-end API and a front-end web-app and then joined together to create the NC-News website. Picture This Place

This is a Full-stack group project. This App is a location based guessing game were a user will go out, take a picture of a building or monument and then post on the app and then another user will look on the app at near-by posted places and go and find the location of the picture, once they're at the location they can make and guess and based on the co-ordinates of the guess the app will give them a medal depening on how close to the location they are. if they're within 25m of the location they will recieve a gold medal, withing 50M will recieve silver and within 75m will recieve bronze.

  1. JavaScript
  2. React Native
  3. Expo/ExpoGo
  4. MongoDB
  5. Mongoose
  6. Node.js
  7. Express.js
  8. Jest
