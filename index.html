NC-News

My NC-News project is a full-stack news application that I developed during my time at Northcoders. Originally, it consisted of two separate projects: a back-end API and a front-end web application. These two components were integrated to create the NC-News website. The NC-News website offers a range of functionalities, including the ability to read and comment on articles, like and delete comments, and post new articles. Additionally, it provides features for sorting articles based on various criteria and allows users to sign in with existing accounts.

Picture This Place

This is a full-stack group project, a location-based guessing game. In this app, a user captures an image of a building or monument, posts it on the app, and challenges other users to locate the pictured place. When a user is at the location, they can submit a guess based on coordinates, and the app awards medals based on their proximity to the actual location. Users who guess within 25 meters receive a gold medal, those within 50 meters get silver, and those within 75 meters earn a bronze medal.
This is a group project consisting of 5 individuals, we each had an equal involvement in designing and creating the app.
